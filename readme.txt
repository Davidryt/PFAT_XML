El archivo champs.rar, contiene imagenes y archivos de configuracion para los html generados por los ficheros xsl. 
A pesar de que la ejecucion de estos no se ve afectada por la ausencia de esta carpeta,
recomendamos extraer la carpeta de manera que la carpeta champs quede al mismo nivel 
que el html (vease Configuracion.jpg)

Para poder realizar la creación de un nuevo campeon es necesario meter el siguiente comando con sus respectivos 
parametros, para ello se necesita:

Para mostrar Información

java MostrarJugadores basejugadores1.xml

Para Insercion de Información:
